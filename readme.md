# 백석새소망교회 썸네일 자동제조 프로그램
- 본 프로그램은, 제 교회 썸네일을 일일이 만들기 귀찮아서 자동화로 만든 프로그램입니다.
- test/open_img.py 파일은 그냥 테스트 삼아서 만든 코드입니다.
- 제 코드 쓸 거면, test/add_txt.py 파일 참고해서 쓰시면 됩니다.
- 라이브러리 사용법 등, gemini의 도움을 받았습니다. 

## 사용법
### dist/add_txt.exe
본 파일을 실행하면 setting.txt 에 세팅된 내용대로 이미지가 출력됩니다.

### setting.txt
변경하고 싶은 정보는 이곳에서 기입 후, 변경하세요. (변수명은 변경하지 마십시오! 시스템 오류를 일으킵니다!)

# 추후
